Projects

Some of the neat things I worked on.
Airport Database

My team, comprising two other individuals and myself, collaborated extensively on designing a sophisticated database tailored for an airline company. Through discussions, we created database tables, considering entities, their corresponding attributes, and the establishment of primary and foreign keys to ensure robust data relationships across tables. Furthermore, we drew up a ER diagrams, and discussed the relationships between entites. This project not only provided a valuable learning experience but also offered an opportunity for meaningful collaboration and professional growth.

Restaurant Management

Me and a partner developed with C# a restaurant mangment system. We used Microsoft Access and WinForms in order to store data for dishes, orders as well as any potenal user. We program and generate a working GUI with winform that could be used for various menus. Using the program, we were able to modify the database we had created and you were able to insert,delete and update certain attributes.
